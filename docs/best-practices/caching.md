## <a name="considerations-for-using-caching"></a><span data-ttu-id="e01a6-146">Considérations sur l’utilisation de la mise en cache</span><span class="sxs-lookup"><span data-stu-id="e01a6-146">Considerations for using caching</span></span>

### <a name="decide-when-to-cache-data"></a><span data-ttu-id="e01a6-148">Décider quand mettre en cache des données</span><span class="sxs-lookup"><span data-stu-id="e01a6-148">Decide when to cache data</span></span>
