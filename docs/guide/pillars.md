<span data-ttu-id="353a5-118">Capacité d’un système à traiter une charge accrue.</span><span class="sxs-lookup"><span data-stu-id="353a5-118">Scalability is the ability of a system to handle increased load.</span></span> <span data-ttu-id="353a5-119">La mise à l’échelle d’une application peut s’effectuer de deux manières :</span><span class="sxs-lookup"><span data-stu-id="353a5-119">There are two main ways that an application can scale.</span></span> <span data-ttu-id="353a5-120">La mise à l’échelle verticale (augmentation de la taille des *instances*) signifie augmenter la capacité d’une ressource, par exemple à l’aide d’une machine virtuelle de plus grande taille.</span><span class="sxs-lookup"><span data-stu-id="353a5-120">Vertical scaling (scaling *up*) means increasing the capacity of a resource, for example by using a larger VM size.</span></span> <span data-ttu-id="353a5-121">La mise à l’échelle horizontale (montée en *charge*) consiste à ajouter de nouvelles instances d’une ressource, telles que des machines virtuelles ou des réplicas de base de données.</span><span class="sxs-lookup"><span data-stu-id="353a5-121">Horizontal scaling (scaling *out*) is adding new instances of a resource, such as VMs or database replicas.</span></span>

<span data-ttu-id="353a5-135">Celle-ci doit être conçue dans le système.</span><span class="sxs-lookup"><span data-stu-id="353a5-135">Horizontal scale must be designed into the system.</span></span> <span data-ttu-id="353a5-136">Par exemple, vous pouvez monter en charge les machines virtuelles en les plaçant derrière un équilibreur de charge.</span><span class="sxs-lookup"><span data-stu-id="353a5-136">For example, you can scale out VMs by placing them behind a load balancer.</span></span> <span data-ttu-id="353a5-137">Toutefois, chaque machine virtuelle du pool doit être en mesure de gérer n’importe quelle requête client.
