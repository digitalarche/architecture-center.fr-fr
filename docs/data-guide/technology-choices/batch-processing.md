<span data-ttu-id="a8853-103">Les solutions de Big Data utilisent souvent de longs traitements par lots pour filtrer, agréger et, plus généralement, préparer les données pour l’analyse.</span><span class="sxs-lookup"><span data-stu-id="a8853-103">Big data solutions often use long-running batch jobs to filter, aggregate, and otherwise prepare the data for analysis.</span></span> <span data-ttu-id="a8853-104">Généralement, ces tâches impliquent de lire les fichiers sources dans un système de stockage évolutif (comme HDFS, Azure Data Lake Store et le Stockage Azure), de les traiter et d’écrire la sortie dans de nouveaux fichiers au sein du système de stockage évolutif.</span><span class="sxs-lookup"><span data-stu-id="a8853-104">Usually these jobs involve reading source files from scalable storage (like HDFS, Azure Data Lake Store, and Azure Storage), processing them, and writing the output to new files in scalable storage.</span></span>
