## <a name="policy-consistency"></a><span data-ttu-id="451d2-130">Stratégie de cohérence</span><span class="sxs-lookup"><span data-stu-id="451d2-130">Policy consistency</span></span>

<span data-ttu-id="451d2-131">Pour s’assurer que les stratégies de gouvernance sont appliquées lors de la création des ressources, une partie de la conception du regroupement des ressources implique l’utilisation d’une configuration commune lors du déploiement des ressources.</span><span class="sxs-lookup"><span data-stu-id="451d2-131">To ensure that governance policies are applied when resources are created, part of resource grouping design involves using a common configuration when deploying resources.</span></span>
